## 题目：油滴扩展

[Link](https://www.luogu.com.cn/problem/P1378)

### 涉及算法/考点：
全排列

### 解题思路：
全排列枚举所有顺序，计算。n很小，全排列复杂度并不会超时，不需要优化
### 解题关键点：
摆放油滴不同的顺序会产生不同的结果，需要列举出全部可能性再逐一比较
### 易错点/出现的错误总结：
1. 计算半径时，可能出现负数，要赋值为0
2. π的取值要到后六位以上（r的上限为$10^6$,$πr^2$要保留整数位）
3. 摆放的第i个油滴的下标并不是[i]
### 通用技巧总结：
1. abs函数可以获取绝对值
2. sqrt函数可以取根号值
